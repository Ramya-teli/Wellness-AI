# Wellness AI - Product Requirements Document

## Project Overview
An AI-powered wellness dashboard combining workout planning, sleep coaching, meditation guidance, and AI chat coaching with Opik observability integration.

## Tech Stack
- **Frontend**: React + TailwindCSS + Framer Motion + Recharts
- **Backend**: FastAPI + MongoDB
- **AI**: Gemini 3 Flash via custom LLM Key
- **Observability**: Opik for LLM-as-judge evaluation

## What's Been Implemented (Jan 27, 2025)

### Core Features
- ✅ Dashboard with wellness score, streaks, progress bars
- ✅ Workout Planner with AI recommendations based on energy level
- ✅ Sleep Coach with AI-powered sleep analysis
- ✅ Meditation Center with AI-generated guided sessions
- ✅ AI Chat Coach with context-aware responses
- ✅ Opik Metrics Dashboard with evaluation tracking

### Opik Integration
- ✅ LLM-as-judge response quality evaluation
- ✅ Safety guardrails checking
- ✅ Experiment tracking by context
- ✅ User feedback collection
- ✅ Quality/relevance/safety metrics visualization
